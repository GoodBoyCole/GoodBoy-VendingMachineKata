Vending Machine Approval Kata - C# version
==========================================

This is starting code for doing approval testing for the [Vending Machine Kata](https://sammancoaching.org/kata_descriptions/vending_machine.html). The idea is to focus on designing the printer and the tests and have them evolve as the production code gains features.
