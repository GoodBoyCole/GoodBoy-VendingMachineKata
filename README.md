# Vending Machine Approval Kata\n\nThis project contains starting code for practicing Approval testing techniques with the [Vending Machine Kata](https://sammancoaching.org/kata_descriptions/vending_machine.html).\n\n## Objectives\n\n1. The initial code includes a functional test for the first feature. Rewrite it to utilize Approval testing.\n2. Write new tests to verify the correct functionality of different coin types.\n3. Implement the "return coins" feature. You can either develop it yourself or merge the existing 'return_coins' branch. Ensure proper testing of this feature and redesign the printer as necessary.\n4. Implement the "select product" feature. Utilize the 'select_product' branch for a shortcut if needed. Create a test for this feature.\n5. Continue developing additional features and their corresponding tests based on the complete kata description.\n\n## Contribution\n\nContributors are encouraged to expand on the project, add tests, improve code quality, and explore Approval testing approaches.\n\n## How to Get Started\n\n1. Clone the repository.\n2. Follow the instructions in the kata description.\n3. Submit improvements via pull requests or issues.\n\n---\n\nHappy coding with Approval testing techniques!
